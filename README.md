# My Custom Website Blocks

A performance-first WordPress block library built from scratch using **React**, **Atomic Design principles**, and the **@wordpress/scripts** build pipeline.

## 🏗 Architecture

This project follows a component-based architecture to ensure UI consistency and code reusability.

* **Atoms (`src/components/`)**: The smallest building blocks (e.g., `NavLink.js`, `ActionButton.js`). These are generic React components.
* **Organisms (`src/blocks/`)**: Full Gutenberg blocks registered in WordPress (e.g., `Hero`, `Site Header`). They consume Atoms to build complex layouts.
* **Build System**: Uses Webpack via `@wordpress/scripts` to compile modern JS/JSX into browser-compatible code in the `/build` directory.
